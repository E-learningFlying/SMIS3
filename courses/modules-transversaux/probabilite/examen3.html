Examen 3 : Projet d'Analyse

Mise en situation : Vous êtes responsable qualité dans une usine de smartphones.

Problème : Défauts d'écran

Le nombre de rayures sur un écran suit une loi de Poisson de paramètre λ = 0.5 par écran.

Chaque écran avec 2 rayures ou plus est jeté (coût : 50€). Les autres sont vendus.

Questions :

  1. Quelle est la probabilité qu'un écran soit jeté ?
  2. Sur une production de 10 000 écrans, quel est le coût des pertes estimé ?

1. Probabilité de rejet :
P(X ≥ 2) = 1 - P(X < 2)=1 - [P(X=0) + P(X=1)].
Loi Poisson : P(k) = e^-0.5 * 0.5^k / k!
P(0) = e^-0.5 ≈ 0.606
P(1) = e^-0.5 * 0.5 ≈ 0.303
P(Rejet) = 1 - (0.606 + 0.303) = 1 - 0.909 = 0.091 (9.1%).

2. Coût :
Nb rejetés = 10 000 * 0.091 = 910 écrans.
Coût = 910 * 50€ = 45 500 €.
