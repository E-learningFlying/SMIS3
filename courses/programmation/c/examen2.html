1. Inversion de tableau

Écrivez une fonction void mirror(int *tab, int size) qui inverse un tableau sur place. 

void mirror(int *tab, int size) {
    for(int i=0; i < size/2; i++) {
        int temp = tab[i];
        tab[i] = tab[size - 1 - i];
        tab[size - 1 - i] = temp;
    }
}
2. Compter les mots

Écrivez int countWords(char *str) qui compte le nombre de mots (séparés par des espaces).

int countWords(char *str) {
    int count = 0;
    int inWord = 0;
    while(*str) {
        if(*str == ' ') inWord = 0;
        else if(!inWord) { inWord = 1; count++; }
        str++;
    }
    return count;
}
3. Fusion de fichiers

Écrivez un programme qui prend f1.txt et f2.txt et concatène le tout dans result.txt.

void copy(FILE *src, FILE *dest) {
    int c;
    while((c = fgetc(src)) != EOF) fputc(c, dest);
}
