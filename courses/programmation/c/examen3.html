Examen 3 : Projet Libre

Le but est de créer une application complète en C.

Sujet A : Bibliothèque

Gestion de livres (Titre, Auteur, ISBN). Ajout, Suppression, Recherche, Sauvegarde fichier.

Sujet B : Mini-RPG Textuel

Création perso, Combat tour par tour, Inventaire, Sauvegarde progression.

Sujet C : Analyseur Log

Lire un fichier de logs, compter les erreurs, filtrer par date, exporter un rapport HTML.

Critères d'évaluation

  • Code propre et indenté
  • Pas de fuites mémoire (Valgrind si possible)
  • Découpage en fonctions / fichiers .h .c
  • Gestion des cas d'erreur (fichiers manquants, saisies invalides)
